# IDENTITÀ DEL MODULO

Il modulo Spese NON è:

❌ simulazione finanziaria  
❌ previsione economica  
❌ sistema bancario  

Il modulo Spese è:

👉 memoria reale del denaro vissuto.

---

# DOMANDA PRINCIPALE

Il modulo deve aiutare a rispondere a:

👉 “Cosa è successo davvero?”

---

# FILOSOFIA FONDAMENTALE

Le Spese rappresentano:

✔ realtà economica vissuta  
✔ movimenti reali  
✔ comportamenti reali  
✔ storico reale  

NON:

✔ simulazioni future

---

# PRINCIPIO STRUTTURALE

# PREVISIONE ≠ REALTÀ

Una spesa prevista NON equivale a:

✔ spesa realmente avvenuta

---

# ESEMPI

## Ricorrenza prevista

Netflix:

17€/mese previsti

NON genera automaticamente:

✔ movimento reale

Il pagamento reale deve poter essere:

- confermato
- modificato
- corretto
- saltato

---

# OBIETTIVO DEL MODULO

Il modulo deve permettere di leggere:

- dove vengono spesi i soldi
- andamento reale delle uscite
- abitudini economiche
- categorie più pesanti
- storico reale famigliare

---

# STRUTTURA FUTURA BASE

Ogni spesa dovrà poter contenere:

✔ importo  
✔ categoria  
✔ persona  
✔ metodo pagamento  
✔ note  
✔ data reale
